Layout rationale

Recommendation: Layout A — Editorial Bento. These shots are product close-ups with intentional framing (not lifestyle full-body, not square mockups). A uniform grid crops the craft out of half of them. Editorial bands let each aspect ratio work and create a page that feels curated, not tiled.

What the photos actually are

  • 15 studio hero shots — extreme close-ups of printed / embroidered apparel (Underdog + Forever Fierce marks).
  • Aspect mix: 10 portrait-dominant (mostly ~3:4 and ~4:5), 5 soft-landscape / near-square detail crops.
  • Job of the image: sell ink quality, type craft, fabric texture — not “people in a gym.”
  • Current /ourwork grid forces square mockups. These new heroes fight that container.

Why A wins

  • Native ratios preserved. Portrait triptychs hold 3:4; landscape band holds the 1.1 crops; no dead letterboxing.
  • Rhythm. Big → three → three detail → color pair → bento → craft close. Eyes don’t go flat.
  • Color pacing. Neutrals open, color breaks mid-page (blue tank / purple USC / Indiana illustration / St. Louis pink), embroidery closes on craft proof.
  • Busy vs quiet. Full graphic prints alternate with left-chest minimal and embroidery so the page breathes.
  • Mobile. Bands collapse to single column (or 2-up for triptychs) without re-authoring the sequence.

Why not B or C as primary

  • B Masonry — fastest to build, respects natural height, but feels like a dump of photos. Weak narrative control; uneven column bottoms; harder captions. Keep as a fallback if you want “more shots later” density.
  • C Runway — luxury look, lots of air. Needs more vertical scroll for 15 shots and can feel sparse on desktop. Better if you cut to 8–10 hero pieces.

Hard layout rules for production

  • Gap: 8–12px (I used 10). Tight = editorial catalog, not Pinterest collage.
  • Corner radius: 2–4px max — or 0. These are streetwear-product, not soft SaaS cards.
  • object-fit: cover only inside a frame that matches the shot’s native ratio. Never cover-crop a 3:4 into a square.
  • Hover: subtle scale (~1.04) + optional caption. No heavy overlays that hide the print.
  • Lightbox on click (you already have this pattern on /ourwork).
  • Background: bone #F2EDE4 or pure white. Dark ink page also works if the whole section goes dark — don’t mix mid-gray.
  • Optional captions: garment / print type only (“Embroidery · hoodie”), not long marketing blurbs.

Suggested final sequence (A)

  • Open: 15 (St. Louis pink, emotional) + 10 (quiet left chest) + 08 (FF badge)
  • Triptych: 01, 03, 11 — classic chest graphics
  • Detail strip: 04, 05, 07 — landscapes / embroidery
  • Color: 09 + 13
  • Bento: 06 large + 02 + 12
  • Close: 14 embroidery craft shot, centered, slightly restrained width

What I would not do

  • Equal square grid (kills half the compositions).
  • Infinite masonry with 40 more mockups mixed in — separate “Templates” from “Hero craft” if both live on the page.
  • Heavy text blocks between every band — one header, one closing CTA is enough.
  • Carousel / slider — these deserve simultaneous comparison and scroll-owned pacing.

If this becomes a full /ourwork rebuild

  • Section 1: this hero craft gallery (the 15).
  • Section 2: existing square mockup library (templates / garment range) in a denser 3–4 col grid below, under a second eyebrow (“More from the archive”).
  • Keep the stats strip if you still want social proof — park it between header and Band 1, or between craft gallery and archive.
